# Neon Data Profile

A futuristic digital networking card built using only HTML5 and CSS3.

This project presents a modern and immersive digital identity card inspired by futuristic dashboards and AI interfaces. The goal of this mini-project was to create a professional networking card without using JavaScript or external frameworks.

---

## 🚀 Features

* Futuristic neon interface
* Animated scan effect
* Responsive modern design
* Skills progress bars
* Professional networking profile
* Pure HTML & CSS

---

## 🛠️ Technologies Used

* HTML5
* CSS3

## 🌐 Project Goal

The objective of this project is to redesign the traditional professional card into a futuristic digital experience adapted for hackathons, networking events and technology forums.

The interface combines modern UI design, neon visual effects and animated components to create an immersive presentation fully developed with HTML and CSS.

---

## 💡 Highlights

This project demonstrates that advanced and visually attractive interfaces can be created without JavaScript by using only:

* HTML structure
* CSS animations
* Modern UI design principles
* Responsive layouts
